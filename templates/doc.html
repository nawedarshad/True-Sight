Introduction

Welcome to the TrueSight AI Image Detection Documentation. This guide provides an overview of our AI image detection technology, algorithms, use cases, and integration instructions. TrueSight is designed to help you identify AI-generated images and ensure the authenticity of visual content.

Algorithms

1. Image Preprocessing

Image preprocessing involves resizing, normalization, and augmentation to prepare the input image for analysis.

Steps:

  • Resizing: Resize the image to a fixed dimension (e.g., 224x224 pixels).
  • Normalization: Normalize pixel values to a range between 0 and 1.
  • Augmentation: Apply data augmentation techniques such as rotation, flipping, etc.

2. Feature Extraction

The algorithm extracts relevant features from the preprocessed image using a convolutional neural network (CNN).

Key Components:

  • Convolutional Layers: Apply filters to detect features such as edges and textures.
  • Pooling Layers: Reduce the spatial dimensions of the feature maps, retaining important information.
  • Fully Connected Layers: Combine the extracted features to form a final feature vector.

3. Classification

The extracted features are fed into a classifier, typically a softmax layer, which outputs the probability that the input image is AI-generated or real.

Steps:

  • Softmax Layer: Calculate the probability distribution over the two classes (AI-generated, real).
  • Thresholding: Use a predefined threshold to make the final classification.

4. Post-Processing and Output

The final step involves post-processing the classification results to provide a user-friendly output.

Steps:

  • Result Interpretation: Convert the classification probabilities into a human-readable format.
  • Visualization: Display the classification result alongside the input image for easy interpretation.
